SigStack

A private AI assistant on Signal, running in a Trusted Execution Environment. Your messages never leave encrypted memory.

User ─→ Signal E2E ─→ [ TEE: Signal CLI ─→ Bot ─→ NEAR AI (GPU TEE) ] └──────── all in encrypted memory ────────┘
Product

What SigStack Can Do

🧠

Conditional Recall

Context-aware memory that recalls past conversations only when relevant — privacy-first, ephemeral by default, with TEE-protected in-memory storage.

🌐

Translate

Real-time translation powered by NEAR AI inference running inside a GPU TEE. Translate messages across languages without exposing plaintext to any operator.

🤖

AI Assistant

Full-featured AI chat with tool use — web search, calculator, weather — all running inside Intel TDX. Cryptographically verifiable with !verify.
